CSS 学习思路

在 HTML 被发明出来时主要用于知识共享（如学术论文），对页面排版要求很简单。刚开始就并没有 CSS 的存在。在那时，HTML 的样式是通过 HTML 添加标签和属性来完成样式的调整。这样做比较麻烦，有了复杂的页面样式需要时，然后 CSS 就应运而生，让内容样式分离。

在学习 CSS 过程中，会发现有许多不正交、不符合直觉的情况出现。如 border 影响 margin 合并，绝对定位会影响 display 为 inline 的元素等情况。但应对日常工作时，我们可以背套路。如水平居中、垂直居中在各种情况下怎么写，两栏布局、三栏布局怎么写。总结好后，遇到相应的情况直接往里面套就 OK

记住几个常见的写法

  • 两种水平居中
  • 三种垂直居中
  • 两种左右结构
  • 两种左中右结构
  • 一行省略
  • 多行省略
