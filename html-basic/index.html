HTML 基础

什么是HTML？

超文本标记语言(Hypertext Markup Lauguage,HTML)是一个可以用结构化 Web 内容并给予其含义和目标的编码语言。

它由一系列的元素（elements）所组成，不同的元素各有功用。

元素

元素由一个开始的 标签 和结束的标签组成，中间用来包含内容
元素的开标前可以带有一些 属性（attribute）

属性：在开标签，提供关于该元素的更多信息，属性由属性名和随后的属性值组成。赋予元素意义和语境。

全局属性：是所有HTML元素共有的属性; 它们可以用于所有元素，尽管属性可能对某些元素没有影响。
全局属性-MDN

块级元素 & 内联元素

  • 块级元素（比如标题和段落）会相互堆叠在一起沿页面向下排列，每个元素分别占一行。
  • 行内元素（比如链接和图片）则会相互并列，只有在空间不足以并列的情况下才会折到下一行显示。
