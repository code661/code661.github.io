当指定条件(condition)内的值为真，if 语句会执行一段语句。如果条件为假，则执行另一段语句。
括号内语句的值，如果是布尔类型则直接判断。但如果是其他类型的值，会被强制转换成布尔类型。转换时调用 ToBoolean，规则如下图：

if(Undefined)

当 if(undenfined) 判断为 false

if(Null)

当 if(null) 判断为 false

if(Boolean)

当括号内为布尔值，直接判断

if(Number)

当括号内的值为数字时，0、NaN 判断为 false ,其他值为 true

if(String)

当括号内的值为字符串类型时，空字符串判断为 false，其他情况都为 true

if(Object)

当括号内是一个对象时，判断为 true
