CSS 布局的套路

认真一点
2018-07-03
CSS

套路：需要支持 IE8 就用 float 布局， 不需要则使用 Flex 布局。
要注意的原则是：

  1. 不到万不得以、不要把宽高写一个固定值
  2. 尽量用高级语法，如 calc、flex
  3. 如果是 IE 都写成固定值

浮动

使用浮动的套路：子元素设置浮动float，父元素清除浮动

平均布局（等分布局）

  • 等分布局是指子元素平均分配父元素宽度的布局方式。
    Tip⭐️：在进行布局前可以画个简图，对子元素的宽度，以及子元素之间的间隙进行简单的计算。让写代码的思路更加清楚

处于每行行首的子元素需要把左 margin 设置为 0，处于行尾的子元素需要把右margin设置为 0。可以利用 nth-child 这个伪类来实现。

但是，由于 nth-child 是 CSS3的属性。在 IE11 以下不可以使用（nth-child - caniuse.com）

可以另外一个负margin 的办法来实现，添加一个 div 包裹子元素，通过设置左右负 margin 来拉伸 div 的宽度。
