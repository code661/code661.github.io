CSS 的 IFC 理解 font-size 、line-height、vertical-align

认真一点
2018-07-03
CSS
  • font-size 指定的是 em-square。印刷的模具大小，
  • line-height 是内联元素的真实占地高度
  • 两个不同字体的兄弟内联元素，当他们两个有相同 font-size 和 line-height 的情况下。为什么会使得他们的父元素高度超出 line-height ？ 因为对齐的问题
  • 图片下面为什么有空隙（以baseline对齐的原因），解决方案：
    1. vertical-align: meddile
    2. img { display: block}
  • 两个 inline-block 元素无法对齐，解决方案： flex 或 float
  • 两个 inline-block 有间隙（间隙产生原因是有换行），解决方案是用flex 或 float
